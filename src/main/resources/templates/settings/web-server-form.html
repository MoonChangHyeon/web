웹/서버 환경설정
서버 및 백엔드에서 사용하는 표준 시간대를 지정합니다.
웹 UI 및 자동 이메일, 로그의 기본 언어를 선택합니다.
시스템에서 사용하는 데이터베이스 종류를 선택합니다.
DB 서버의 호스트명을 입력합니다.
DB 서버의 접속 포트입니다. MariaDB 기본값: 3306
DB 접속에 사용할 계정명입니다.
DB 접속에 사용할 비밀번호입니다.
접속 및 운영에 사용할 데이터베이스 이름을 입력합니다.
분석결과(FPR, PDF, XML 등) 및 업로드파일 저장 위치를 지정합니다.
시스템 동작 및 오류 발생시 기록되는 로그의 상세 수준을 지정합니다.
분석/시스템 로그파일이 저장되는 위치를 지정합니다.
분석 요청시 업로드할 수 있는 파일 최대 용량을 지정합니다.
보고서 자동메일 전송 등 이메일 기능을 사용할지 설정합니다.
이메일 송신용 SMTP 서버 주소를 입력합니다.
SMTP 서버 포트를 입력합니다.
메일 발송에 사용되는 이메일 주소(예: noreply@company.com)
